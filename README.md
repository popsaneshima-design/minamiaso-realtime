# 南阿蘇鉄道 リアルタイム運行ボード

南阿蘇鉄道の沿線案内イラスト上で、現在時刻に合わせて列車が実際のダイヤどおりに走るリアルタイム運行ボードです。

🌐 **公開URL**: https://popsaneshima-design.github.io/minamiaso-realtime/

スマホからもそのまま閲覧できます。

---

## ✨ 主な機能

### 🚃 リアルタイム運行アニメーション
- 現在時刻と公式ダイヤから各列車の現在位置を計算し、イラスト上を移動
- 1秒ごとに位置を更新（CSS transition で滑らかに補間）
- 路線の曲線部分（立野→長陽 / 見晴台→高森）は経由点を設定し、実際の線路に沿って走行
- 進行方向に応じて電車の頭が左右反転

### 🚂 車両デザイン
- **普通列車**: 青系の旅客車スタイル（窓・ドア・ヘッドライト・4輪）
- **トロッコ列車**: 赤の蒸気機関車スタイル（ボイラー・キャブ・煙突・大型駆動輪）
- 走行中は軽くバウンスアニメーション

### 📍 駅ホバーで次便表示
- 各駅マーカーにマウスを乗せると、その駅から次に発車する列車を上下方向ごとに表示
- 種別（普通／乗入／トロッコ）・発車時刻・残り時間を一目で確認
- 営業終了後は「本日の運行は終了しました」と表示

### 🎉 リアルタイム トースト
- 列車が出発・各駅到着・終点到着した瞬間にポップアップ通知
- 駅ごとの遊び心ある一言（例: 白川水源「お水汲んでいかれませ〜」）
- 5.4秒経過で自動フェードアウト

### 🎪 季節列車の自動判定
- トロッコ「ゆうすげ号」と一部普通列車（8002・8003）の運行日を自動判定
  - 3/14〜11/30 の土休日
  - 春期(3/20〜4/5、4/29〜5/6) / 夏期(7/18〜8/23、8/27〜28) / 秋期(9/3〜25 の特定日)
- 運休日は対象列車を非表示

### 🔍 表示切替
- 「近い列車のみ」: 走行中＋出発10分前以内の列車
- 「全列車表示」: 当日のすべての列車（季節列車除く運休日は隠す）

## 🎨 デザイン

地図イラストの自然色に合わせたパレット：
- 背景: 空色 → 抹茶 → 砂色のグラデーション
- 上り電車: 深青 / 下り電車: 中青
- 汽車: 朱赤
- アクセント: 山吹・大地・森緑

## 📊 データソース

- **ダイヤ**: 南阿蘇鉄道 2026年3月14日改正の時刻表
- **路線図イラスト**: [南阿蘇鉄道公式サイト](https://www.mt-torokko.com/railwayguide/img/illust_ensenkanko.jpg)
- **駅座標**: 公式イラストの赤点を画像解析（OpenCV/scipy）で検出した実ピクセル座標

## 🛠 技術スタック

- **シングルHTMLファイル**（外部ビルド不要）
- React 18（CDN umd）
- Babel Standalone（JSX のブラウザ内変換）
- SVG で路線図オーバーレイ・列車アイコン描画
- ホスティング: GitHub Pages

## 🔄 更新方法

`index.html` を編集してプッシュすると、数分後に GitHub Pages へ反映されます。

```bash
cd ~/Desktop/minamiaso-realtime
# index.html を編集
git add index.html
git commit -m "更新内容のメッセージ"
git push
```
